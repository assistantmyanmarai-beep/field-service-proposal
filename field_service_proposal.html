Overview
EIREE Water Purifier ၏ Installation & Field Service Team (Electrician/Plumber co-partners) အတွက် AI-powered guidance system တစ်ခု တည်ဆောက်ပေးပါမည်။ Technician တစ်ယောက်ချင်းစီ customer အိမ်ရောက်တဲ့အခါ installation steps၊ troubleshooting၊ customer-facing explanation တွေအတွက် AI ဆီကို real-time consult လုပ်နိုင်ပြီး၊ Client က technician တစ်ယောက်ချင်းစီရဲ့ work history ကို customer/job အလိုက် ခွဲခြားပြီး review လုပ်နိုင်အောင် system ဖန်တီးပေးပါမည်။
Two-System Architecture
Field Service Assistant Technician
Job/Customer အလိုက် Topic ခွဲထားသော Group Chat
  • Job/Customer တစ်ခုချင်းစီအလိုက် Topic သီးခြားစီ — history မရောစေရန်
  • Installation step-by-step guide (AI)
  • Photo/Video ပို့ပြီး "ဒါမှန်လား" မေးနိုင်ခြင်း
  • Knowledge Base အပေါ်အခြေခံပြီး တိကျသော လမ်းညွှန်ပေးခြင်း
  • Customer ကို ရှင်းမပြတတ်တဲ့အခါ AI ဆီ ပြန်မေးနိုင်ခြင်း
  • "Job Complete" ရိုက်ထည့်ရင် Topic auto-archive
Admin Management Tool Client/Owner
Private Chat — Client/Owner Only
  • Order Entry — Job အသစ် ထည့်သွင်းခြင်း (guided, button-based)
  • Knowledge Base Add — Product installation guide ထည့်ခြင်း
  • Knowledge Base Edit — ပြင်ဆင်ခြင်း
  • Reports — Job status overview ကြည့်ခြင်း
Workflow — Order to Completion
1
Order Entry — Admin Tool ကနေ Order အသစ် ထည့် (Customer name, phone, product, address)
2
Job Code — System က Job Code auto-generate (ဥပမာ: JOB-2026-001)
3
Topic Creation — Field Assistant ထဲမှာ Topic auto-create
4
Job Assignment — Technician — assigned job list ကြည့်ပြီး Topic ထဲ ဝင်
5
AI Consultation — Technician — AI ဆီ installation guide မေး၊ photo/video ပို့ပြီး verify ခံ
6
Completion — Job ပြီးရင် "JOB COMPLETE" ရိုက်
7
Auto-Archive — Topic auto-archive, status update
8
Client Review — Client — completed job history ကြည့်နိုင်
AI Capability — Knowledge Base Driven
  • Product တစ်ခုချင်းစီအတွက် ကြိုတင်ပြင်ဆင်ထားသော installation guide၊ pipe/connection type၊ common mistakes တို့ကို Knowledge Base အဖြစ် training ပေးထားမည်
  • Technician ပို့သော photo/video ကို Knowledge Base ထဲက reference image များနှင့် နှိုင်းယှဉ်ပြီး feedback ပေးမည်
  • AI က ရှင်းရှင်းလင်းလင်း မမြင်ရပါက ခန့်မှန်း မဖြေဘဲ — ပိုနီးကပ်စွာ ထပ်မံ ဓာတ်ပုံ/ဗီဒီယို ရိုက်ပေးရန် တောင်းဆိုမည် (Accuracy-first design)
Note — EIREE Water Purifier သည် Zero Electricity system ဖြစ်သောကြောင့် main risk category မှာ pipe/water connection accuracy ဖြစ်ပါသည်။

Note — Dashboard အကြောင်း
Phase 1 တွင် visual Dashboard ကို ထည့်သွင်းမထားသေးပါ — Budget သတ်မှတ်ချက် ပိုမိုတိကျစွာ ဆွေးနွေးပြီးမှသာ Phase 2 အဖြစ် ထည့်သွင်းရန် စဉ်းစားထားပါသည်။

Dashboard မပါသော်လည်း — Telegram-based system ကို Client အတွက် အဆင်ပြေအောင်၊ Job/Order list ကို လွယ်ကူစွာ ကြည့်ရှုနိုင်အောင် ဒီဇိုင်းထုတ်ပေးထားပါသည်။ Admin Tool တစ်ခုတည်းကနေ Order ထည့်ခြင်း၊ Knowledge Base Manage ခြင်း၊ Report ကြည့်ခြင်း — အားလုံး လုပ်ဆောင်နိုင်ပါသည်။

Dashboard ထည့်သွင်းလိုပါက ထပ်ဆောင်း setup cost ရှိမည်ဖြစ်ပြီး — Phase 2 အနေနဲ့ သီးခြား quote ပေးပါမည်။
